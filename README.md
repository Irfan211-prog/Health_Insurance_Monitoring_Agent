# Health Insurance Monitoring Agent

## Project Overview
This project aims to detect potentially fraudulent health insurance claims using machine learning techniques.

## Dataset
- Beneficiary data
- Inpatient claims
- Outpatient claims
- Provider fraud labels

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Project Structure
```
Datasets/
Notebooks/
Preprocessing/
Tools/
Conclusions/
```

## Techniques used:
1. Data Imbalance Handling: SMOTE and ADASYN
2. ML and Deep Learning Models: Decision Trees, Random Forests, LightGBM and XGBoost
3. Pearson Correlation Model and Regression Models
